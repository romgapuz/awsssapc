This tool is intended for use only to provide cost estimate. It does not guarantee correctness and accuracy of the data and calculation provided. The following assumptions are observed:

  • EC2 selection based on SAPS chooses the nearest SAPS match
  • EC2 selection based on CPU/RAM prioritizes RAM over CPU. CPU count are used and performance are not evaluated.
  • EBS (GP2) storage calculation is based on the following formula: RAM x 3 + Root + USR + DB Data + DB Log
  • EBS (ST1) and S3 storage calculation is based on the following formula: Root + DB Data + DB Log
  • Snapshot storage calculation is based on the following formula: Root + USR
  • EC2 instance/SAPS table uses the list from https://aws.amazon.com/sap/instance-types/. The list is further filtered on the EC2 instance types available on the selected region.

What's New!

  • Region selection - you can now select a region on the drop-down menu on the top-right corner in the navigation bar.
  • Regular pricing update - EC2 on-demand and reserved instance pricing are regularly updated. There's a background job running on AWS Lambda that pulls pricing data daily and generate json files for use by this calculator.
